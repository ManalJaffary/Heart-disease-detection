# Heart-disease-detection
.This project presents a comprehensive Heart Disease Detection System that leverages both Machine Learning (ML) and Deep Learning (DL) techniques to predict whether a patient is at risk of heart disease based on clinical data. The objective is to develop an intelligent and reliable predictive model that can assist healthcare professionals in making early diagnostic decisions, ultimately improving patient outcomes and reducing the risk of severe cardiovascular complications.
The project uses a publicly available heart disease dataset containing various medical attributes such as age, sex, chest pain type, resting blood pressure, cholesterol level, fasting blood sugar, electrocardiographic results, maximum heart rate, exercise-induced angina, ST depression, and other clinical features. Before model training, the dataset undergoes preprocessing steps including data cleaning, feature scaling, label encoding, and train-test splitting to ensure high-quality input for the learning algorithms.
Multiple machine learning algorithms are implemented and evaluated, including K-Nearest Neighbors (KNN) and Gaussian Naïve Bayes, alongside a Multi-Layer Perceptron (MLP) deep neural network built using TensorFlow/Keras. Each model is assessed using standard performance metrics such as Accuracy, Precision, Recall, and F1-Score, enabling a fair comparison of their predictive capabilities. Training time is also recorded to evaluate computational efficiency.
The project is developed in Python using libraries such as NumPy, Pandas, Scikit-learn, TensorFlow/Keras, Matplotlib, and PyTorch. Data visualization techniques and performance evaluation help provide insights into model behavior and classification effectiveness.
This repository demonstrates the complete machine learning workflow—from data preprocessing and feature engineering to model training, evaluation, and result analysis. It serves as an educational resource for students, researchers, and developers interested in applying artificial intelligence to healthcare. The project highlights the potential of AI-powered diagnostic systems to support medical professionals through accurate, data-driven decision-making and represents a practical step toward intelligent healthcare solutions.
